## 📋 Sobre o Projeto
Este projeto faz parte da disciplina de BI, onde exploramos o potencial do Python para processamento de grandes volumes de dados e aplicação de modelos de IA.

## 🚀 Como Executar
1. Certifique-se de ter o Python instalado.
2. Instale as dependências necessárias:
   ```bash
   pip install pandas
   ```
3. Execute o arquivo principal:
   ```bash
   python funcoes.py
   ```

## 📁 Estrutura de Arquivos
- `funcoes.py`: Script inicial com funções de teste.
- `README.md`: Documentação do projeto.
