# 📊 Análise de Salários Iniciais por UF - Python Pandas

Este projeto consiste em um script Python que utiliza a biblioteca **Pandas** para processar e analisar dados de salários iniciais de diferentes estados brasileiros (UFs). O objetivo é extrair insights estatísticos como médias salariais, rankings e agrupamentos por região.

## 🚀 Funcionalidades

O script realiza as seguintes operações:
- Identifica o estado com o **menor** e o **maior** salário.
- Localiza a posição específica do estado de **Goiás** no ranking.
- Calcula a **média salarial geral** de todos os estados.
- Realiza agrupamentos por **Região** para mostrar:
    - Maior e menor salário por região.
    - Média salarial por região.
- Identifica as regiões com a maior e a menor média salarial do conjunto.

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Pandas**: Para manipulação e análise de dados.
